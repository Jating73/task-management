## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Setup

```bash
# Install NestJS CLI
$ npm i -g @nest/cli

# Create a new project
$ nest new project-name

# Create a module
$ nest g module moduleName

# Create a controller
$ nest g controller controllerName --no-spec

# Create a service
$ nest g service serviceName --no-spec
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
